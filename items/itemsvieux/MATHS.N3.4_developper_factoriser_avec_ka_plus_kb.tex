%\titreitem{Développer ou factoriser une expression littérale à l'aide de $k(a+b)=ka+kb$.}

A) Développer, puis réduire, les expressions suivantes :

\begin{multicols}{3}

\begin{enumerate}

\item $ A = 6(3x-5) $ 
\item $ B = 7x(4-2x) $ 
\item $ C = 4(x-5)+x(2x-8) $ 

\end{enumerate}

\end{multicols}

B) Factoriser les expressions suivantes :

\begin{multicols}{3}

\begin{enumerate}

\item $ D = 14x^2+21x $ 
\item $ E = 3(2x-5) - 7x(2x-5) $ 
\item $ F = 6x-9x^2$ 

\end{enumerate}

\end{multicols}
